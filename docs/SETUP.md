# Arkpad Setup Guide

## Prerequisites

- Node.js LTS
- npm or pnpm
- Git

## Local Setup

1. Clone repository.
2. Install dependencies.
3. Run type check and tests.
4. Start Arkpad app.

## Suggested Scripts

- install: install dependencies
- dev: run local development app
- build: compile packages
- test: run automated tests
- lint: run linter
- typecheck: run TypeScript checks

## Environment Checklist

- [ ] Correct Node version installed
- [ ] Package manager available
- [ ] Dependencies installed successfully
- [ ] Tests pass locally
